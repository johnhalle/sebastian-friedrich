# CHANGELOG v31

Stand: 08. Juni 2026

## Änderungen gegenüber v30

- Lange Unterseiten-Titel werden nun bei Bedarf automatisch verkleinert, damit sie in einer Zeile bleiben. Dies betrifft insbesondere `Datenschutz`.
- Startseite / Neueste Werke: Die graue Kartenfläche wird auf Desktop von der Bildunterkante bis in den Textbereich durchgezogen. Die weißen Zwischenräume zwischen den Karten bleiben erhalten.
- Sonderregel für `Marco`: Das Bild erhält unten eine dezente Trennlinie zur grauen Kartenfläche, weil der Bildhintergrund dem Kartenhintergrund nahekommt.
- Cache-Busting für CSS/JS auf `v=31` erhöht.
- ZIP-Paket verschlankt: Es werden nur noch die tatsächlich eingebundenen lokalen Inter-Webfonts mitgeliefert.

## Beibehalten

- Mobile Darstellung der Startseiten-Werkkarten bleibt ohne graue Kartenfläche.
- Keine zusätzlichen Kartenrahmen oder Schatten.
